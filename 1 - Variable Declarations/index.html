1 - Variable Declarations

We have 3 ways of declaring variables in javascript:

  • Using let
  • Using const
  • Using var

Let's dive into the snippet:

var name = 'Jhon'; let lastName = 'Doe'; const age = '21';

So, what would happen if we try to reassign the values?Open your Chrome Dev Tools

name = 'Jane'; lastName = 'Doll'; age = '12';
