2 - Strict Mode

The "use strict" defines that JavaScript code should be executed in "strict mode".

Tip: The "use strict" directive is only recognized at the beginning of a script or a function.
Tip 2: Click here for more detailed info about the strict mode.

What would happen if we run those snnipets bellow?

  • Undeclared variable assignment in strict mode

    'use strict'; x = 'This is just a test'; console.log(x);
  • Undeclared variable assignment not in strict mode

    y = 'This is just another test'; console.log(y);
