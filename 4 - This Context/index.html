4 - This context

The "this" in vanilla javascript on browser environment is just an alias for the window variable.

Assuming this page already have this javascript code lodaded:

var myTest = 'This is a context test';

Because of hoisting (recently learned) the var is hoisted to the global space.
In other words, the variable myTest will be available on both places: this.myTest and window.myTest.

Go ahead and test it on your console!
